# Probability & Bayes Theorem Notes

## What is Probability?

Probability is the measure of how likely an event is to happen.

* Value ranges from 0 to 1
* 0 = impossible event
* 1 = certain event

### Example

* Probability of getting a head when tossing a fair coin = 0.5

---

## What is Conditional Probability?

Conditional probability is the probability of an event occurring given that another event has already occurred.

It helps us update our understanding when we have additional information.

### Example

* Probability that a person has a fever given that they are sick.

---

## What Does P(A|B) Mean?

P(A|B) is read as:

"Probability of A given B"

It represents the probability that event A occurs after knowing that event B has already occurred.

### Example

* A = Student passes an exam
* B = Student studied regularly

P(A|B) means the probability that the student passes the exam given that the student studied regularly.

---

## Key Takeaways

* Probability measures uncertainty.
* Conditional probability uses additional information.
* P(A|B) means the probability of A given B.
* Conditional probability is the foundation of Bayes' Theorem.
* Bayes' Theorem helps update beliefs based on new evidence.
