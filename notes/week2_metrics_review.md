# Week 2 Metrics Review

## Metrics Covered

### Accuracy

Measures overall prediction correctness.

### Precision

Measures how many predicted positives are actually positive.

### Recall

Measures how many actual positives are correctly identified.

### F1 Score

Balances Precision and Recall.

### ROC-AUC

Measures classification performance across thresholds.

### Confusion Matrix

Provides TP, TN, FP, and FN counts.

## Key Takeaway

Model evaluation should be aligned with business goals rather than relying only on accuracy.
