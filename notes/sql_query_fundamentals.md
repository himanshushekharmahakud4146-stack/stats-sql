# SQL Query Fundamentals

## SELECT

Used to retrieve data from a table.

Example:

SELECT * FROM sales;

---

## WHERE

Used to filter records based on a condition.

Example:

SELECT * FROM sales
WHERE revenue > 7000;

---

## ORDER BY

Used to sort data.

Ascending:

ORDER BY revenue;

Descending:

ORDER BY revenue DESC;

---

## LIMIT

Used to restrict the number of rows returned.

Example:

SELECT * FROM sales
LIMIT 3;

---

## Key Learning

SELECT retrieves data.

WHERE filters data.

ORDER BY sorts data.

LIMIT controls the number of rows returned.
